# How to run

```bash
java -DAPP_REDIS_HOST=redis_host \
     -DAPP_DB_HOST=db_host \
     -DAPP_DB_USER=db_user \
     -DAPP_DB_PASSWORD=db_password \
     -jar cursoaws.jar
```
